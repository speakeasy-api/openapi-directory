# integratedPrograms

## Overview

APIs to be published by the Health ID system for benefit linkage

### Available Operations

* [createHealthIdByDemoAuthUsingPOST](#createhealthidbydemoauthusingpost) - Create health id using Aadhaar Demo Auth.
* [createHealthIdByMobileUsingPOST](#createhealthidbymobileusingpost) - Create health id using mobile Authentication.
* [delinkHidBenefitUsingPOST](#delinkhidbenefitusingpost) - De-Linked with hid.
* [findByAadharUsingPOST](#findbyaadharusingpost) - Search health id number using aadhar or aadhar token.
* [findByHealthIdUsingPOST](#findbyhealthidusingpost) - Search benefit using health id number.
* [generateAadharOTPUsingPOST2](#generateaadharotpusingpost2) - Generate Aadhaar OTP on registrered mobile number
* [generateMobileOtpUsingPOST](#generatemobileotpusingpost) - Generate mobile OTP on registrered mobile number
* [linkHidBenefitUsingPOST](#linkhidbenefitusingpost) - Linked with hid.
* [notifyBenefitUsingPOST](#notifybenefitusingpost) - Create health id using notify Benefit.
* [updateAccountInformationUsingPOST1](#updateaccountinformationusingpost1) - Update account information

* [updateStatusUsingPOST](#updatestatususingpost) - Update health id status .
