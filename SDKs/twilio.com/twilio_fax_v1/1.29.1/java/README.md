### SDK SDK

* `fetchFax` - Fetch a specific fax.
* `fetchFaxMedia` - Fetch a specific fax media instance.
* `listFax` - Retrieve a list of all faxes.
* `listFaxMedia` - Retrieve a list of all fax media instances for the specified fax.
