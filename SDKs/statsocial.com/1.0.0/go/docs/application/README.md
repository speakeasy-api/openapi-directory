## GetApplicationsStatus

Remaining window and remaining count details, also returns reports that have been generated by this account.


### Example Usage

```go
package main

import(
	"context"
	"log"
	"openapi"
	"openapi/pkg/models/operations"
)

func main() {
    s := sdk.New(
        sdk.WithSecurity(shared.Security{
            APIKey: "YOUR_API_KEY_HERE",
        }),
    )

    ctx := context.Background()
    res, err := s.Application.GetApplicationsStatus(ctx, operations.GetApplicationsStatusRequest{
        Key: "provident",
    })
    if err != nil {
        log.Fatal(err)
    }

    if res.ApplicationStatus != nil {
        // handle response
    }
}
```
