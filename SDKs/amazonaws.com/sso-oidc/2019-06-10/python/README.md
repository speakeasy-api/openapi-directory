## SDK Available Operations

### SDK SDK

* `create_token` - Creates and returns an access token for the authorized client. The access token issued will be used to fetch short-term credentials for the assigned roles in the AWS account.
* `register_client` - Registers a client with AWS SSO. This allows clients to initiate device authorization. The output should be persisted for reuse through many authentication requests.
* `start_device_authorization` - Initiates device authorization by requesting a pair of verification codes from the authorization service.
